## **How to install and run locally ?**

```
$ git clone https://github.com/jagrut-sharma/Baat-cheet.git
$ cd Baat-cheet
$ npm install
$ npm run dev
```

## **Features -**

- User Feed

  - List all user posts
  - Sort by Date & Trending

- Create a Post
- Like a post
- Edit & Delete Post
- User Profile

  - Add Avatar
  - Add Bio
  - Add Portfolio URL

- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Authentication
  - Sign-up
  - Login
  - Logout
- Upload Images while creating a post
- Add Emoji while creating a post
- Edit User Profile Photo
- Full Responsive design
- Comments CRUD Operations

## **Built with -**

- React JS
- React Context API + useReducer
- React Router v6.4
- Tailwind CSS
- Headless UI
- Radix UI
- Consists of real backend
